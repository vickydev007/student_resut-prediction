# student_resut-prediction
Works in a team project. I played a ML engineer role to predict course and result outcomes.

1. Trained a model to predict result and implemented logics to fetch course level weakness and course status.
2. Use joblib to save my model and load my trainned model.
3. Fetch only the required features and scale the data to maintain variation.
4. Use the Scaled data and use it to predict the outcomes.
5. Map the numeric prediction to categorical data to make it understand my Human easily.
6. return the results in FastAPI endpoint that can be used for advanced purpose(private process).
